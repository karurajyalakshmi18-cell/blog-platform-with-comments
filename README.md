# Blog Platform with Comments

## Overview
A full-stack blogging platform where users can register, log in, create blog posts, edit posts, delete posts, and interact through comments.

## Features
- User Registration
- User Login & Authentication
- Create Blog Posts
- Edit Blog Posts
- Delete Blog Posts
- Comment System
- Responsive Design
- RESTful API Integration
- Database Connectivity

## Technology Stack

### Frontend
- HTML
- CSS
- JavaScript
- React.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB
